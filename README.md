# Spam Message Detector

## Project Overview
Spam Message Detector is a Machine Learning based web application that identifies whether a given message is **Spam** or **Safe (Ham)**.  
It uses Natural Language Processing (NLP) techniques and a trained ML model to classify messages.

---

##  Features
- Detects Spam and Safe messages
- Shows prediction with probability (%)
- Smart detection for suspicious links
- Simple and user-friendly web interface
- Real-time message analysis

---

##  Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- TF-IDF Vectorizer
- HTML & CSS

---

##  Machine Learning Model
- Algorithm: Naive Bayes (MultinomialNB)
- Text Processing: TF-IDF (Term Frequency - Inverse Document Frequency)
- Dataset: SMS Spam Collection Dataset
